# 🗺️ Интерактивная карта с метками


## 📋 Описание

**Интерактивная карта с метками** — это мощный веб-редактор для создания интерактивных карт с возможностью добавления кастомных меток, тегов и экспорта готового просмотрового файла.

### ✨ Возможности

  - **Загрузка карты** — поддержка изображений в форматах JPG и PNG
  - **Добавление меток** — загрузка собственных иконок (PNG, JPG, SVG) для создания уникальных маркеров
  - **Система тегов** — создание, редактирование и фильтрация меток по тегам 
  - **Интерактивное перемещение** — перетаскивание меток из меню на карту
  - **Редактирование меток** — изменение названия, тега, описания и размера (правый клик по метке)
  - **Экспорт/Импорт** — сохранение всей карты с метками в один HTML-файл
